**Prerequisites to run from Source:**
- Windows 7+
- Python 2.7
- Pyinstaller
  - pip install pyinstaller
- Pywin32
  - pip install pypiwin32
- Pycrypto
  - pip install pycrypto
- wxPython
  - pip install -U wxPython
  
**Prequisites for running Binary:**
- Windows 7+
- Python 2.7
- Pyinstaller
